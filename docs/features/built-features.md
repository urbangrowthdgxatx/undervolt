Transformed a basic permit analysis pipeline into a **production-ready infrastructure mapping platform** with ML clustering, interactive visualizations, time series forecasting, and semantic search capabilities.

### 6. Data Processing Scripts

#### `scripts/name_clusters.py` ⭐ NEW
Auto-generates descriptive cluster names based on keyword analysis.
```python
# Input: cluster_names.json (IDs only)
# Output: cluster_names.json (with names)

{
  "0": {
    "name": "New Residential Construction",
    "size": 23100,
    "top_keywords": [...]
  }
}
```

Rule-based naming logic:
- Demolition > 35% → "Demolition Projects"
- Foundation > 80% → "Foundation Repairs"
- New > 80% + Residential → "New Residential Construction"
- etc.

#### `scripts/prepare_geojson.py` ⭐ NEW
Converts permit CSV to GeoJSON for maps.
```python
# Input: permit_summary_by_zip.csv
```
